void veml6030_i2c_open ( void  )

Opens I2C specifically for the VEML 6030.

Initializes an I2C_OPEN_STRUCT with the specifications of the VEML6030. This maps out the locations for the SCL and SDA and also creates a callback for the scheduler. I2C open is then called, starting the process of opening i2c for the VEML6030
