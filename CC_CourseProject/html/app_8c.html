void app_letimer_pwm_open ( float  period,
float  act_period,
uint32_t  out0_route,
uint32_t  out1_route 
)
static

Initializes LETIMER with struct values.

This function first creates a struct of type APP_LETIMER_PWM_TypeDef and then initializes all of the parameters inside said struct with the passed in values. The struct is then passed into the letimer_pwm_open function and counting is started with letimer_start

Note
Called in the app_letimer_setup function.
Parameters
[in]periodDefines the period length of PWM in seconds, floating point variable.
[in]act_periodDefines the active period of PWM in seconds, floating point variable.
[in]out0_routeDefines the output of 0 to the gpio port/pin, routing register.
[in]out1_routeDefines the output of 0 to the gpio port/pin, routing register.
